# eBay Automation with Playwright

## 🎯 Project Objective
A portfolio project demonstrating end-to-end (E2E) automation using Playwright and TypeScript. This suite tests core functionalities of the eBay Australia marketplace.

## 🛠️ Tech Stack
- **Framework:** Playwright
- **Language:** TypeScript
- **CI/CD:** GitHub Actions
- **Locators:** User-facing (Role, Placeholder)

## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/aj4tig/ebay-playwright-automation.git`
2. Install dependencies: `npm install`
3. Run tests in UI mode: `npx playwright test --ui`

## ✅ Test Cases Covered
- [x] Basic Search functionality (Himalayan 450)
- [x] Strict-mode button handling
